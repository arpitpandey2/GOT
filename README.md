# Summoning the Dev Server
*(a.k.a. "How do I actually run this?")*

You'll need **Node.js** installed.

Think of it as becoming a maester.

Without it, you're basically holding a sword by the wrong end.

```bash
npm install
npm run dev
```

Then open the localhost link Vite gives you.

Congratulations.

You're officially Lord Commander of your browser.

If the dev server decides to behave like Bran and disappear into another dimension:

```bash
npx vite
```

Usually works.

Unlike the Small Council.
